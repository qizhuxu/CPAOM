# Product Overview

CPA 账号管理系统 (CPAOM) - A multi-deployment CPA (ChatGPT Proxy Account) management system.

## Core Purpose

Manage ChatGPT proxy accounts across multiple servers with automated token revival, batch operations, and usage monitoring.

## Three Deployment Options

1. **Shell Version** - Command-line tool for personal use and automation
2. **Cloudflare Version** - Zero-maintenance cloud deployment with web UI
3. **Web Version** - Self-hosted Docker/local deployment with full control

## Key Features

- Multi-server CPA management
- Account listing and status monitoring
- Batch usage checking
- Automatic OAuth token revival (refresh expired tokens)
- Batch download/upload of authentication files
- Account enable/disable management
- Statistics and reporting
- Scheduled maintenance tasks (Cloudflare & Web versions)
- Data backup and synchronization

## Target Users

- Individual users managing personal CPA accounts
- Teams requiring collaborative account management
- Organizations needing centralized CPA infrastructure

## Primary Use Cases

- Monitor account usage across multiple CPA servers
- Automatically revive expired OAuth tokens to maintain availability
- Batch download authentication files for backup
- Batch upload authentication files for migration or restoration
- Disable accounts after failed token revival attempts
- Track statistics and operational history
